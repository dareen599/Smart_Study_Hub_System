#  Smart Study Hub System

##  Description
An intelligent academic collaboration platform designed to bridge the gap between students by matching them based on field of study and academic level to enhance peer-to-peer learning and project collaboration.

##  Key Features
* **Smart Matching**: Automatically connects students with similar academic interests and specialties.
* **Level Synchronization**: Matches users based on their current academic year to ensure compatible collaboration.
* **Field-Specific Focus**: Designed specifically for technical disciplines like Computer Science and Software Engineering.

##  Built With
* **Version Control**: Git & GitHub
* **Target Group**: University Students
